# 💸 Daily Expense Tracker (MERN)

A full-stack **expense management web app** that helps users track spending, manage income, and monitor savings goals.

---

## 🚀 Live Demo

👉 Live Demo: https://daily-expense-log.vercel.app
👉 API: https://daily-expense-log.onrender.com

---

## 🧠 Features

* 🔐 User Authentication (JWT based login/register)
* ➕ Add, view, and delete expenses
* 📊 Track total spending & category breakdown
* 💰 Manage monthly income and savings goal
* 🌙 Dark/Light mode toggle
* 📁 Export expenses as CSV
* 📱 Responsive UI (mobile-friendly)

---

## 🛠 Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication

### Deployment

* Frontend: Vercel
* Backend: Render

---

## ⚠️ Known Limitations

* ⏳ Initial delay on login due to backend cold start (Render free tier)
* 📶 Requires internet for API calls
